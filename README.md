# Order-Processor-Java-Application
An advanced OOP assignment involving processing orders, customers, and more using an existing database.

Given an existing database, I had to create the system to read in the information and store it in classes and records. This involved efficient communications across the system and complex OOP techniques including multiple inheritance hierarchies, overriding, formatting, and more.

This project helped me develop my skills in OOP and understand the fundamental techniques to ensure good quality code.
